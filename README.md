# 🛍️ Brechó da Lucy

Uma vitrine digital simples e elegante para organizar e exibir produtos.  
Este projeto inclui:
- Login e painel administrativo
- Banner dinâmico
- Bloco de contato com WhatsApp
- Filtros de busca por estado e qualidade
- Vitrine com cards numerados mostrando **nome, preço, estado, qualidade e descrição**

---

## 🚀 Tecnologias
- **HTML** para estrutura
- **CSS** para estilo e responsividade
- **JavaScript** para lógica de filtros, login e renderização dinâmica
- **Supabase** para banco de dados
- **Cloudinary** para hospedagem de imagens

---

## 📸 Demonstração
Cada produto aparece em um card padronizado, com imagem ajustada em tamanho fixo, preço destacado e informações completas.
